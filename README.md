CodeTech Task 3 - Dashboard Development
Objective
To create an interactive dashboard for data visualization using Python.

Tools Used
Python
Pandas
Plotly
Dash
Dataset
Restaurant Tips Dataset (seaborn tips dataset)

Features
Interactive dropdown filter (Day-wise analysis)
Bar chart: Total bill by gender
Pie chart: Smoker distribution
Dynamic updates based on user selection
Key Insights
Males generally have higher total bills than females.
Non-smokers form the majority across all days.
Sunday shows highest restaurant activity.
Output
Fully functional interactive dashboard using Dash framework.
